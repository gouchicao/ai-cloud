## 附件
* Micro SD卡（每个主机需要一张）
* Raspberry Pi 4（树莓派4）电源适配器   5V4A
* Jetson Nano电源适配器   5V3A
* Micro HDMI 转 HDMI 高清线（Raspberry Pi 4）
* HDMI 转 HDMI 高清线（Jetson Nano）
* 散热片（Raspberry Pi 4）
* 网线（Jetson Nano）
* 铜柱（可以用来将多个树莓派4安装到一起）
* 插排
